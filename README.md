# つーりんく（TouLink）

## サイト概要
### サイトテーマ
ツーリング仲間を探したり、イベントを企画・参加できる**バイク好きのためのコミュニティサイト**です。
​
### テーマを選んだ理由
私はバイクに乗ることが趣味で、休日にはよくツーリングに出かけます。
しかしながら、ツーリング仲間を見つけることは意外と難しく、SNSで募集をかけてもドタキャンやマナー違反が発生したり、
「景色を楽しみながらゆっくり走りたい」「目的地に早く着きたい」など、ツーリングに求める価値観の違いから、相性の良いメンバーを見つけにくいという課題がありました。
これらの悩みは、初心者ライダーやリターンライダーに特によく見られる問題でもあります。

そこで、同じ価値観や目的を持つバイク好き同士が安心して交流し、ツーリングイベントを安全かつスムーズに企画・運営できるサービスがあれば便利だと考えました。
このような背景から、本テーマでポートフォリオを制作することにいたしました。
​
### ターゲットユーザ
- ツーリング仲間を探している人
- バイクイベントやツーリングを企画したい人
- 安心できる環境で交流したいバイクコミュニティを探している人
- 初心者ライダーで、経験者からアドバイスをもらいたい人
​
### 主な利用シーン
- ツーリングの開催や参加者を募集したいとき
- 自分のツーリング写真やルートを共有したいとき
- 気になるライダーをフォローしたり、メッセージを送りたいとき
- 他のライダーの投稿を参考にして新しいルートや仲間を見つけたいとき
